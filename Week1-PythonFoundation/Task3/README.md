## File Descriptions

### 1. `bank_account.py`
This file contains a **Bank Account System** implemented using OOP concepts.

**Features:**
- Base class `BankAccount`
- Child class `SavingsAccount`
- Deposit and withdraw functionality
- Encapsulation using protected variables
- Polymorphism by overriding methods
- Usage of magic methods for initialization and printing objects

**Concepts Practiced:**
- Classes and Objects  
- Inheritance and Polymorphism  
- Encapsulation  
- Magic Methods  

---

### 2. `decorators.py`
This file demonstrates how **decorators** work in Python.

**Features:**
- Custom execution time decorator
- Measures how long a function takes to run
- Demonstrates function wrapping

**Concepts Practiced:**
- Decorators  
- Function wrapping  
- Python internal function calls  

---

### 3. `generators.py`
This file focuses on **generators and iterators**.

**Features:**
- Fibonacci number generator
- Custom range generator
- Uses `yield` instead of `return`

**Concepts Practiced:**
- Generators  
- Iterators  
- Memory-efficient loops
