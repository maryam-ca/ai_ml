# Task 1 – Python Programming & Git Basics

## Overview
This task is part of Week 1 of the AI/ML Fellowship program. The purpose of this task is to build a strong foundation in Python programming while learning the basics of Git and GitHub for version control.

## Task Objectives
The main objectives of this task are:
- To understand basic Python programming concepts
- To practice writing simple and beginner-friendly Python programs
- To learn how to use Git and GitHub in a professional manner
- To organize project files and write proper documentation

## Learning Outcomes
Through this task, I learned:
- Core Python concepts such as variables, data types, input and output, loops, and conditional statements
- How to write, run, and test simple Python programs
- How to create a GitHub repository and push code using Git commands
- The importance of maintaining a clean folder structure and documentation

## Python Programs Included
This task includes the following Python programs:
- area_of_circle.py – Calculates the area of a circle
- factorial.py – Finds the factorial of a given number
- String_reverse.py – Reverses a given string
- number_guessing_game.py – A simple number guessing game using random numbers

## How to Run the Programs
Follow the steps below to run the Python programs:
1. Open the terminal or command prompt
2. Navigate to the Task1 folder
3. Run the required Python file using the command:

python filename.py

### Example
python area_of_circle.py

Completing this task helped me strengthen my Python basics and understand how to manage projects using GitHub. It also taught me how to write clean code, organize files properly, and submit assignments in a professional way.
